Step-by-Step Flow

Each phase is designed to be explainable and safe for decision support.

1. Data Intake

Profile, lifestyle, symptoms, family history, and lab text.

2. Marker Extraction

TSH, T3, T4, HbA1c, insulin, cortisol, cholesterol, glucose.

3. Risk Scoring

Rule engine runs first, ML inference runs if enabled and available.

4. Explanation

Trigger signals and AI summary for clinical interpretation.

5. Reporting

Suggested tests, recommended actions, and structured JSON output.

6. Dashboard

User history, admin analytics, exports, and audit trail.

Model Strategy

Rule engine guarantees deterministic outputs, while ML models refine scores when enabled.

  1. Rules first: consistent baseline scoring.
  2. ML optional: used only when models are available.
  3. AI summary: optional, powered by your OpenAI key.

Outputs

  • Risk percentages and levels
  • Key trigger signals
  • AI-generated explanation
  • Suggested tests and actions
